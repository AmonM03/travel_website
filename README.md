## Requirements

**Have these requirements downloaded on your machine:**

- Git

- Node.js

- npm

**Cloning the Repository**

```shell
git clone https://github.com/AmonM03/travel_website.git
```

```shell
cd travel_website
```

**Installation**

```shell
npm install
```

**Running the project**

```shell
npm run dev
```
